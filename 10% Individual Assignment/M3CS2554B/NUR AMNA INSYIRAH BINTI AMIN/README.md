# INTRODUCTION 
This project "Classification of Large-Scale Numeric Dataset" is developed as an individual assignment for the Network Programming. The application utilizes Python to classify a massive dataset of 5,000,000 numeric records into specific categories based on complex mathematical logic. 

# OBJECTIVE 
To demonstrate the performance advantages of parallel processing over traditional sequential execution. 

# PROBLEM STATEMENT 
A standard single-threaded application only utilizes one logical processor, leaving the rest of the CPU hardware idle. This results in longer execution times for CPU-bound tasks such as numeric classification. This project addresses the inefficiency by implementing "Multiprocessing" to distribute the workload accross multiple CPU cores, effectively reducing processing time.
